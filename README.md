# English Workshop
An English learning website


## Getting Started

Note: this is small project. Not for sale or commercialization


### Installing

Install [NodeJS] (https://nodejs.org/en/)
Open project by Visual Studio Code
Open terminal
Run command
```
npm install
```

Then run
```
npm start
```
